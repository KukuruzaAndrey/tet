## Terminal tetris in 50 programming languages

### Quick run
```
make
```
`ctrl+q` for quit

### Font
This game looks nicer with special font [Square](https://strlen.com/square/) made by Wouter van Oortmerssen.
.ttf file in assets folder.

### Set language to run
```
make PL=C   #JS, etc
```

### Done languages
- [x] C
- [x] JS
- [x] JAVA

### Test
```
make test PL=C
```
